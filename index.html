Pasante de Ingeniería en ITAM

Creatividad artíficial: imitación de estílos músicales
Objetivo

Explorar la creatividad artificial para la generación automatizada de melodías imitando estílos de obras musicales publicadas.

Descripción

El análisis frecuencial de una señal auditiva proporciona información sobre los desplamientos entre sus notas musicales. Apoyandose de las librerías de MATLAB para análisis de señales e ingenieria, se sistematiza el análisis de una melodía descomponiendola en sus componentes frecuenciales y modelando su frecuencia estadística mediante un proceso de Markov. El modelo estadístico resultante, al usarse en un algorítmo de agrupamiento como "Random Tree Forest" sirve de base para clasificar otras muestras fuera del grupo de entrenamiento.

Partiendo de un sistema con una clasificación aceptable. Se pretende implementar un algorítmo genético para generar una nueva melodía fuera del grupo de entrenamiento con un grado de clasificación aceptable. En teoría esta nueva melodía contendría componentes frecuenciales semejantes a los del grupo de entrenamiento.
